Tahapan Integrasi Snap Woocommerce

  1. Instalasi Plugin
  2. Tes Transaksi


  • Tahap 1Instalasi Plugin
  • Tahap 2Tes Transaksi

Apa yang Diperlukan

  1. Wordpress minimal v3.9.1 (sudah di tes hingga v4.5.x).
  2. WooCommerce minimal v2.1.11 (sudah di tes hingga v2.5.x).
  3. PHP minimal v5.4.
  4. MySQL minimal v5.0.

  5. Plugin Midtrans untuk WooCommerce [ Github | Zip ].

    Info: Info:
    Silahkan melihat halaman berikut untuk daftar plugin & library lebih lengkap.

Proses Instalasi

  1. Download dan ekstrak plugin dari repositori ini.
  2. Ekstrak plugin yang baru di unduh, rubah nama folder modul menjadi midtrans-woocommerce.
  3. Upload folder plugin ke direktori wp-content/plugins/ pada instalasi Wordpress Anda.
  4. Install dan aktifkan plugin melalui menu plugin pada admin panel Wordpress Anda.

  5. Masuk ke menu WooCommerce - Settings - Checkout - Midtrans, masukan data-data yang diperlukan.

    • Isi Title dengan text yang akan ditampilkan kepada pelanggan.
    • Pilih Environment. Sandbox untuk test transaksi dan Production untuk transaksi asli.
    • Isi Client and Server key. Anda dapat menemukan ini di Midtrans MAP Dashboard.


  6. Login ke portal MAP untuk mengubah isi pada beberapa opsi berikut:

    URL Role Redirect URL
    Payment Notification URL http://[your web]/?wc-api=WC_Gateway_Midtrans
    Finish Redirect URL http://[your web]/?wc-api=WC_Gateway_Midtrans
    Error Redirect URL http://[your web]/?wc-api=WC_Gateway_Midtrans
    Unfinish Redirect URL http://[your web]/?wc-api=WC_Gateway_Midtrans


Note: Note:
Silahkan melihat halaman Wiki Snap-Woocommerce untuk tutorial lainnya.
